二进制中1的个数(剑指Offer-16)#

题面#

实现 pow(x, n) ，即计算 x 的 n 次幂函数（即，xn）。不得使用库函数，同时不需要考虑大数问题。

示例#

示例 1：

1输入：x = 2.00000, n = 10
2输出：1024.00000

示例 2：

1输入：x = 2.10000, n = 3
2输出：9.26100

示例 3：

1输入：x = 2.00000, n = -2
2输出：0.25000
3解释：2-2 = 1/22 = 1/4 = 0.25

限制#

  • -100.0 < x < 100.0
  • -2^31 <= n <= 2^31-1
  • -10^4 <= x^n <= 10^4

思路#

简简单单快速幂。

代码#

 1class Solution {
 2public:
 3    double myPow(double x, int n) {
 4        double res = 1.0;
 5        long long m = n;
 6        if(m<0){
 7            m=-m; x=1.0/x;
 8        }
 9        while(m){
10            if(m&1)
11                res = res*x;
12            x = x*x;
13            m >>= 1;
14        }
15        return res;
16    }
17};
