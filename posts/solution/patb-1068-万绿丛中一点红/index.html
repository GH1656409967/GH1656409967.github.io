万绿丛中一点红 (PATB-1068)#

题面#

对于计算机而言，颜色不过是像素点对应的一个 24 位的数值。现给定一幅分辨率为 M×N 的画，要求你找出万绿丛中的一点红，即有独一无二颜色的那个像素点，并且该点的颜色与其周围 8 个相邻像素的颜色差充分大。

输入#

输入第一行给出三个正整数，分别是 M 和 N（≤ 1000），即图像的分辨率；以及 TOL，是所求像素点与相邻点的颜色差阈值，色差超过 TOL 的点才被考虑。随后 N 行，每行给出 M 个像素的颜色值，范围在 [0,2^24) 内。所有同行数字间用空格或 TAB 分开。

输出#

在一行中按照 (x, y): color 的格式输出所求像素点的位置以及颜色值，其中位置 x 和 y 分别是该像素在图像矩阵中的列、行编号（从 1 开始编号）。如果这样的点不唯一，则输出 Not Unique；如果这样的点不存在，则输出 Not Exist。

代码#

 1const int mxn = 1e3 + 5;
 2int dir[][2] = {
 3    {-1,-1}, {-1, 0}, {-1, 1},
 4    { 0,-1}, { 0, 1},
 5    { 1,-1}, { 1, 0}, { 1, 1},
 6};
 7int n, m, tol, a[mxn][mxn];
 8
 9bool ok(int x, int y)
10{
11    for(int i=0; i<8; i++)
12    {
13        int tx = x+dir[i][0];
14        int ty = y+dir[i][1];
15        if(tx < 0 || tx >= n) continue;
16        if(ty < 0 || ty >= m) continue;
17        if(Abs(a[x][y] - a[tx][ty]) <= tol)
18            return 0;
19    }
20    return 1;
21}
22
23int main()
24{
25    scanf("%d %d %d", &m, &n, &tol);
26    map<int, int> mmp;
27
28    for(int i=0; i<n; i++)
29    {
30        for(int j=0; j<m; j++)
31        {
32            scanf("%d", &a[i][j]);
33            mmp[a[i][j]]++;
34        }
35    }
36
37    int di, dj, num = 0;
38    for(int i=0; i<n; i++)
39    {
40        for(int j=0; j<m; j++)
41        {
42            if(ok(i, j) && mmp[a[i][j]]==1){
43                num++;
44                if(num > 1) break;
45                di = i, dj = j;
46            }
47        }
48        if(num > 1) break;
49    }
50
51    if(num==0)
52        printf("Not Exist\n");
53    else if(num==1)
54        printf("(%d, %d): %d\n", dj+1, di+1, a[di][dj]);
55    else
56        printf("Not Unique\n");
57
58    return 0;
59}
