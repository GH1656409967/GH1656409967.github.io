一、准备#

卸载已存在的MySQL

1# rpm -qa | grep -i mysql

1# rpm -e --nodeps PackageName

于MySQL官网根据需要下载，参考链接：

  • mysql80-community-release-el7-3.noarch.rpm
  • mysql-connector-java-8.0.16-1.el7.noarch.rpm

二、MySQL#

配置#

更新MySQL的yum源

1# rpm -ivh mysql80-community-release-el7-3.noarch.rpm

yum安装

1# yum install mysql-community-server

测试#

启动MySQL服务

1# systemctl start mysqld.service

查看MySQL的初始密码

1# grep 'password' /var/log/mysqld.log

登录

1# mysql -uroot -p

修改密码

1mysql> ALTER USER 'root'@'localhost' IDENTIFIED BY 'new password';

使用简单密码则需要先执行以下两条

1mysql> set global validate_password_policy=0;
2mysql> set global validate_password_length=1;

exit退出后即可使用新密码重新登录

附上: 关闭MySQL服务

1# systemctl stop mysqld.service

MySQL的yum源移除

1# yum remove mysql80-community-release-el7-3.noarch.rpm

三、JDBC#

配置#

将 mysql-connector-java-8.0.16-1.el7.noarch.rpm 解压，找到 mysql-connector-java.jar，将jar置于java的lib目录下/ext中。以本机为参考，目录是：/usr/lib/jvm/jre/lib/ext/

测试#

写一个简单测试 test.java如下，编译运行查看结果即可。

 1import java.sql.*;
 2public class test {
 3    public static void main(String[] args) {
 4        try {
 5            //Class.forName("com.mysql.jdbc.Driver");
 6            String url = "jdbc:mysql://localhost";
 7            Connection con = DriverManager.getConnection(url, "root", "password");
 8            if (con != null) {
 9                System.out.println("success...");
10                con.close();
11            }
12            else {
13                System.out.println("failure!!!");
14            }
15        }
16        catch (ClassNotFoundException e) {
17            e.printStackTrace();
18        }
19        catch (SQLException e) {
20            e.printStackTrace();
21        }
22    }
23}
